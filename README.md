WebSocketサーバーとクライアントのサンプル001
===============================

使い方
-------

1. このプロジェクトを `git clone` するか、`dist/ws-server.js` のみをダウンロードします。
2. 以下のコマンドを実行します。

    ```
    $ node dist/ws-server.js
    ```

3. ウェブブラウザで、`http://localhost:8080` にアクセスします。
